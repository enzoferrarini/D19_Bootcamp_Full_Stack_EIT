Consideraciones

- Los artículos se encunetran almacenados en un Json Server https://my-json-server.typicode.com/ que recoge infomarción de unas Base de Datos generada en GitHub. Las busquedas ejecutan API FETCH.

- Tener en cuenta que para realizar busquedas a través Json Server, la busqueda debe ser exacta caso contrario no devolverá resultado alguno. Esta es una limitación del tipo de SERVER utilizado. Para evitar esta situación, y si bien no la forma más eficiente, en una primera instancia se descargan todos los productos, y las búsquedas se realizan sobre un array local obtenido desde:
https://my-json-server.typicode.com/enzoferrarini/jsonFiles/electro
